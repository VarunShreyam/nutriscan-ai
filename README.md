## ✨ Features

- 🔍 **Instant Nutrition Analysis** — type any food name and get a full breakdown in seconds
- 📷 **Image Recognition** — upload a food photo and let AI identify and analyze it
- 🌍 **190+ Cuisines** — from Dal Rice to Bibimbap to Shawarma, it knows them all
- 🔥 **Calories & Macros** — protein, carbs, fat, fiber with visual progress bars
- 💊 **Micronutrients** — iron, calcium, vitamin C, sodium, potassium with daily value %
- ⭐ **Health Score** — every food rated out of 10
- 🔐 **Privacy First** — uses your own Anthropic API key, data never touches any external server

## 🚀 Getting Started

### 1. Clone the repo
```bash
git clone https://github.com/YOUR_USERNAME/nutriscan-ai.git
cd nutriscan-ai
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the app
```bash
npm run dev
```

### 4. Get your API key
- Go to [console.anthropic.com](https://console.anthropic.com)
- Sign up and create an API key
- Paste it into the app when prompted

## 📸 How It Works

1. Enter your Anthropic API key (stored in browser only)
2. Type a food name or upload a photo
3. Get instant detailed nutrition info

## 🔐 Privacy

Your API key is stored **only in your browser** and is never sent to any server other than Anthropic's official API.
